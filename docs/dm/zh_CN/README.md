DM (Data Migration) 用户文档
===

如果是第一次使用 DM，建议从 [Get Started](./get-started.md) 开始。

如果想更详细地了解 DM 的使用、限制甚至是原理，可以参考以下 index 直接阅读感兴趣的部分。


### 简介

DM (Data Migration) 是一体化数据同步任务管理平台，支持全量备份和 MariaDB/MySQL binlog 增量同步，设计的主要目的是
   - 标准化 （例如:工具运行、错误定义）
   - 降低运维使用成本
   - 简化错误处理流程
   - 提升产品使用体验
